## Install me

```
python -m pip install itkdb-browser
```

## Run me

```
itkdb-browser
```

or

```
python -m itkdb_browser
```

The login screen will pick up the same variables that [itkdb][itkdb-link] picks
up.

[itkdb-link]: https://pypi.org/project/itkdb/
